# عميل محلي لتحويل EPUB إلى AZW3

أداة مرئية تعمل محلياً لتحويل ملفات EPUB إلى AZW3 / Kindle KF8 باستخدام Calibre. تدعم السحب والإفلات، التحويل الدفعي، عرض التقدم، سجل التحويل، التنزيل الفردي، وتنزيل عدة ملفات داخل ملف ZIP.

## معاينة الواجهة

![واجهة EPUB إلى AZW3](docs/interface-preview.png)

## الميزات

- تحويل EPUB إلى AZW3 محلياً
- رفع دفعي بالسحب والإفلات أو اختيار الملفات
- قائمة ملفات تعرض الاسم ووقت الرفع والنوع والحالة
- عرض التقدم الكلي
- سجل تحويل بتصميم يشبه Kindle
- تنزيل ملف واحد أو عدة ملفات داخل ZIP
- اختيار مجلد الإخراج
- دعم macOS و Windows

## المتطلبات

- Python 3
- Calibre

macOS:

```zsh
brew install --cask calibre
```

Windows:

```bat
winget install --id calibre.calibre -e
```

## التشغيل

macOS: انقر مرتين على `start.command`.

Windows: انقر مرتين على `start.bat`.

## طريقة الاستخدام

1. اسحب ملفات EPUB إلى اللوحة اليسرى أو انقر لاختيارها.
2. حدد الملفات التي تريد تحويلها.
3. اختياري: اختر مجلد الإخراج.
4. انقر على `تحويل المحدد`.
5. نزّل ملفات AZW3 من سجل التحويل.

تبقى الملفات على جهازك. لا يستطيع Calibre افتراضياً تحويل ملفات EPUB المحمية بـ DRM.
